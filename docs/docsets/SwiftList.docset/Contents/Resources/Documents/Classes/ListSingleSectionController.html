ListSingleSectionController

public final class ListSingleSectionController : ListSectionController

This section controller is meant to make building simple, single-cell lists easier. By providing the type of cell, a closure to configure the cell, and a closure to return the size of a cell, you can use an ListAdapter-powered list with a simpler architecture.
