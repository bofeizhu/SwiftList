ListAdapterUpdateListener

public protocol ListAdapterUpdateListener : AnyObject

Conform to this protocol to receive events about ListAdapter updates.

  • listAdapter(_:didFinishUpdateOfType:animated:)

    Notifies a listener that the listAdapter was updated.

    Declaration

    Swift

    func listAdapter(_ listAdapter: ListAdapter, didFinishUpdateOfType updateType: ListAdapterUpdateType, animated: Bool)

    Parameters

    listAdapter

    The ListAdapter that updated.
    updateType

    The type of update executed.
    animated

    A flag indicating if the update was animated. Always false for ListAdapterUpdateType.reloadData.
